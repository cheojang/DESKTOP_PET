# 🐱 나만의 사진으로 만드는 데스크톱 펫 (시메지 v2.0)

사용자가 업로드한 인물(아이, 가족 등)이나 반려동물의 사진에서 **얼굴만 자동으로 쏙 인식하여 추출**하고, 귀엽고 앙증맞은 **아기 고양이 몸통 스프라이트들과 실시간으로 결합(조립)**하여 바탕화면 위를 노닐게 하는 윈도우용 데스크톱 가젯 프로그램입니다.

---

## 🌟 주요 모션 및 기능 소개

* **아장아장 걷기 (Walk)**: 발을 번갈아 버둥버둥 움직이며 아기자기하게 좌우로 걸어 다닙니다.
* **쌔근쌔근 눕기 (Sleep)**: 바닥에 얌전히 앉아 쉬다가 지치면 대자로 엎드려 꿀잠을 잡니다. 자는 동안 **머리가 위아래로 미세하게 움직이며 쌔근쌔근 숨 쉬는 호흡 애니메이션**이 작동합니다.
* **벽에 꼭 매달리기 & 빼꼼 (Peek)**: 걷다가 화면 왼쪽/오른쪽 벽에 다다르면 화면 너머로 몸을 숨긴 채 벽을 꼬옥 붙잡고 사용자를 빼꼼 훔쳐봅니다. 가끔 작업 표시줄 밑으로 쏙 숨기도 합니다.
* **마우스 드래그 & 중력 낙하**: 펫을 마우스 왼쪽 버튼으로 잡고 붕 띄워 끌고 다닐 수 있으며 (이때 대롱대롱 발버둥을 칩니다), 마우스를 놓으면 중력 가속도를 받으며 작업 표시줄 바닥 위로 스르륵 떨어집니다.
* **실시간 사진 교체 (Hot-Swapping)**: 프로그램이 실행되는 중에도 시스템 트레이 아이콘을 우클릭하여 **"사진 변경"** 메뉴를 누르면, 즉시 새로운 사진 속 얼굴로 조립되어 하늘에서 툭 떨어집니다!

## 🛠️ 개발 환경 및 기술 스택

* **언어**: Python 3.x
* **GUI 프레임워크**: PyQt5 (테두리 및 배경이 100% 투명한 최상단 오버레이 윈도우 구현)
* **컴퓨터 비전**: OpenCV (Haar Cascade 기반 전면 얼굴 탐지 엔진 탑재)
* **이미지 처리**: Pillow (PIL) (투명 레이어 마스킹 작업 및 실시간 목 조인트 기준 이미지 조립 연산)

## 💻 설치 및 실행 방법

초보자분들을 위해 처음부터 실행하는 단계를 친절하게 안내합니다.

### 1. 개발 환경 설치 및 가상환경 세팅
터미널(윈도우 PowerShell 또는 CMD)을 열고 아래 명령어를 순서대로 입력해 주세요.

```powershell
# 1) 프로젝트 폴더로 이동 (D드라이브 경로 설정 시)
d:
cd "d:\VibeCoding\DESKTOP_PET (2606)"

# 2) 독립된 개발용 가상환경(.venv) 생성
python -m venv .venv

# 3) 가상환경에 필요한 패키지들(PyQt5, OpenCV, Pillow 등) 설치
.\.venv\Scripts\pip install PyQt5 pillow opencv-python rembg "rembg[cpu]"
```

### 2. 귀여운 몸통 스프라이트 자산 빌드
고양이 몸통 자산(PNG 파일 5장)을 자동으로 생성하기 위해 아래 스크립트를 1회 실행합니다.
```powershell
.\.venv\Scripts\python.exe create_assets.py
```

### 3. 프로그램 실행
바탕화면에 펫을 소환합니다!
```powershell
.\.venv\Scripts\python.exe desktop_pet.py
```

## 🎮 조작법 및 사용 팁

* **펫 이동**: 마우스 왼쪽 버튼을 꾹 누른 채로 원하는 위치로 끌고 다닙니다.
* **메뉴창 열기**: 화면 우측 하단 윈도우 시계 옆(숨겨진 아이콘 서랍 `^`)에서 고양이 모양 아이콘을 찾아 **마우스 우클릭**을 합니다.
  * **사진 변경 (새 펫 등록)**: 내 아이나 반려동물 얼굴 사진을 넣으면 실시간 얼굴 오리기 작업이 시작됩니다. (얼굴이 똑바로 찍힌 정면 사진일수록 얼굴이 더 이쁘게 오려집니다.)
  * **종료**: 펫 프로그램을 안전하게 종료하고 바탕화면에서 해제합니다.
